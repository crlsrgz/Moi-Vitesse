# Moi

## ViteJS Boilerplate for ReactJS

## Installation

Clone this repo and install using npm

```bash
npm i
```

## Usage

### Development

```bash
npm run dev
```

### Production

```bash
npm build
```

## Development Feaures

- TailwindCSS
- EsLint
- Prettier
