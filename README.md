# Moi

## ViteJS Boilerplate for ReactJS

## Installation

Clone this repo and install using npm

```bash
npm i
```

## Usage

```bash
npm run dev
```

```bash
npm build
```

## Development Feaures

- TailwindCSS
- EsLint
- Prettier
